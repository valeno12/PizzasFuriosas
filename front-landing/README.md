# front-landing

Landing publica de Pizza Furiosa hecha con Astro.

## Comandos

```sh
npm install
npm run dev
npm run build
```

## Variables de entorno

```env
PUBLIC_API_URL=http://localhost:5054/api
PUBLIC_WHATSAPP_PHONE=5493434537186
PUBLIC_PHONE_DISPLAY="+54 9 3434 53-7186"
```

`PUBLIC_WHATSAPP_PHONE` se usa para el link de WhatsApp y puede escribirse con o sin signos; la app lo convierte a solo digitos para `wa.me`.
